# Simple Calculator
 
A terminal-based calculator written in Python. Enter two numbers and an operator and the program returns the result. Handles invalid input, invalid operators, and division by zero gracefully.
 
## How to Run
 
```bash
python app.py

or

python3 app.py
```
 
## How It Works
 
1. You're prompted for two numbers and an operator
2. The result is calculated and printed
3. Invalid input loops back and prompts you again
